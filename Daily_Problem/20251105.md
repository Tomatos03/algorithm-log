# [E - Romantic Glasses](https://codeforces.com/contest/1915/problem/E)

## 解题思路

1. 原问题可以转换为: 找到一个子数组, 子数组中奇数和等于偶数和
2. 奇数和等于偶数等价于偶数和加上奇数和的相反数等于0或奇数和加上偶数和的相反数等于0 ,  所以可以将原数组的奇数位(或偶数位)上的数, 取相反数
3. 对于一个确定的子数组右端点,  如果存在$pre\_sum[r] = pre\_sum[l - 1]$, 那么回答为$Yes$, 否则为$No$

## 参考代码


```cpp
\\ 时间复杂度O(n)
#include <bits/stdc++.h>
using namespace std;

#define endl '\n'
#define rep(i, a, b) for (int i = a; i <= b; ++i)
#define erp(i, a, b) for (int i = a; i >= b; --i)

using i32 = int;
using i64 = long long;
using pii = pair<int, int>;
using pll = pair<i64, i64>;

const int mod = 1000000007, mod0 = 998244353;

void solve() {
    int n;
    cin >> n;
    vector<int> a(n + 1);
    rep (i, 1, n) {
        cin >> a[i];
        if (i & 1)
            a[i] = -a[i];
    }
    vector<i64> pre_sum(n + 1);
    rep (i, 1, n)
        pre_sum[i] = pre_sum[i - 1] + a[i];

    set<i64> st;
    st.insert(0);

    rep (i, 1, n) {
        if (st.count(pre_sum[i])) {
            cout << "YES" << endl;
            return;
        }
        st.insert(pre_sum[i]);
    }
    cout << "NO" << endl;
}

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int t;
    cin >> t;
    while (t-- > 0)
        solve();
    return 0;
}
```
