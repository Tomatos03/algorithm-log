# [1018. 可被 5 整除的二进制前缀](https://leetcode.cn/problems/binary-prefix-divisible-by-5/)

## 解题思路

1. 注意到从$num[0\cdots i] \rightarrow nums[0 \cdots i + 1]$有如下变化
   $$
   v0 = nums[0] * 2^i + nums[1] * 2^{i - 1} + \cdots + nums[i] * 2^0 \rightarrow \\ 
   v1 = v0 * 2 + nums[i + 1] * 2^0
   $$
2.   由取模的运算规律可以得到: $v1 \% 5 = ((v0 \% 5 * 2) \%5 + nums[i + 1]) \%5$

## 参考代码


```cpp
// 时间复杂度O(n)
class Solution {
public:
    vector<bool> prefixesDivBy5(vector<int>& nums) {
        vector<bool> ans;
        int pre_v = 0;
        for (int &v : nums) {
            pre_v = ((pre_v << 1) % 5 + v) % 5;
            ans.push_back(pre_v == 0);
        }
        return ans;
    }
};
```
