# [3512. 使数组和能被 K 整除的最少操作次数](https://leetcode.cn/problems/minimum-operations-to-make-array-sum-divisible-by-k/)

## 解题思路

1. 给定数组的和取余$k$得到的数就是最少操作次数, 因为每一次操作仅能够减少$1$

## 参考代码


```cpp
// 时间复杂度O(n)
using i64 = long long;
class Solution {
public:
    int minOperations(vector<int>& nums, int k) {
        int sum = 0;
        for (int &v : nums) {
            sum += v;
        }
        sum %= k;
        return sum;
    }
};
```
