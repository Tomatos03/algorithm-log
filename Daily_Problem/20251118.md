# [717. 1 比特与 2 比特字符](https://leetcode.cn/problems/1-bit-and-2-bit-characters/)

## 解题思路

1. 注意到如果当前比特位是$1$, 那么无论下一位是什么都可以跳过(满足题目跳条件2), 即跳过下一个比特位和当前比特位; 如果当前比特位是$0$跳过当前比特位
2. 让$i = 0$从比特数组起始开始模拟$1$中的过程, 由于题目保证最后一位比特位必然是$0$, 在最后只需要判断$i$是否为$n - 1$

## 参考代码


```cpp
// 时间复杂度O(n)
class Solution {
public:
    bool isOneBitCharacter(vector<int>& bits) {
        int n = bits.size(), i = 0;
        while (i < n - 1) {
            i += bits[i] + 1;
        }
        return i == n - 1;
    }
};
```
