# [2589. 完成所有任务的最少时间](https://leetcode.cn/problems/minimum-time-to-complete-all-tasks/)

## 解题思路

1. 将给定数组按照右端点进行升序排序
1. 想要减少完成任务的时间, 需要尽量晚的运行电脑. 因为对于给定区间$[l, r]$, 下一轮遍历要么和当前区间没有交集, 要么有交集并且交集为$[k, r]$. 

## 参考代码


```cpp
// 时间复杂度O(n^2)
class Solution {
public:
    int findMinimumTime(vector<vector<int>>& tasks) {
        sort(tasks.begin(), tasks.end(), [&](auto &x, auto &y){
            return x[1] < y[1];
        });
        int cnt = 0, n = tasks.size();
        vector<bool> run(2001, false);
        for (int i = 0; i < n; ++i) {
            int l = tasks[i][0], r = tasks[i][1], duration = tasks[i][2];
            int run_cnt = 0;
            for (int j = l; j <= r; ++j) {
                run_cnt += run[j];
            }
            duration -= run_cnt;
            while (duration > 0) {
                if (!run[r]) {
                    run[r] = true;
                    ++cnt;
                    --duration;
                }
                --r;
            }
        }
        return cnt;
    }
};

```
