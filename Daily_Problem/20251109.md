# [2169. 得到 0 的操作数](https://leetcode.cn/problems/count-operations-to-obtain-zero/)

## 解题思路

思路一

1. 输入数据量为$10^5$, 每次减少1, 至多执行$2 \times 10 ^5$次, 可以直接模拟

思路二

1. 注意到当$a > 2b$, 这个时候$a$需要减多次$b$, 可以通过一次计算$a \% b$计算出多次减法之后$a$的值, 此时的次数贡献为$\lfloor \frac{a}{b} \rfloor$

## 参考代码


```cpp
// 时间复杂度O(n)
class Solution {
public:
    int countOperations(int num1, int num2) {
        int cnt = 0;
        while (num1 != 0 && num2 != 0) {
            if (num1 >= num2)
                num1 -= num2;
            else
                num2 -= num1;
            ++cnt;
        }
        return cnt;
    }
};
```

```java
// 时间复杂度O(log n)
using i64 = long long;
using pii = pair<int, int>;
class Solution {
public:
    int countOperations(int num1, int num2) {
        int cnt = 0;
        while (num1 > 0 && num2 > 0) {
            if (num1 > num2) {
                cnt += num1 / num2;
                num1 %= num2;
            } else {
                cnt += num2 / num1;
                num2 %= num1;
            }
        }
        return cnt;
    }
};

```
