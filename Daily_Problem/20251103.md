# [2257. 统计网格图中没有被保卫的格子数](https://leetcode.cn/problems/count-unguarded-cells-in-the-grid/)

## 解题思路

1. 相邻且颜色相同的气球, 仅保留移除气球需要时间最长的那个气球

## 参考代码


```cpp
\\ 时间复杂度O(n)
using i64 = long long;
using pii = pair<int, int>;
class Solution {
public:
    int minCost(string colors, vector<int>& neededTime) {
        int cnt = 0;
        int n = colors.size(), i = 0;
        while (i < n) {
            int j = i, k = -1, sum = 0;
            while (i < n && colors[i] == colors[j]) {
                if (k == -1 || neededTime[k] < neededTime[i]) {
                    k = i;
                }
                sum += neededTime[i];
                ++i;
            }
            cnt += sum - neededTime[k];
        }
        return cnt;
    }
};

```
