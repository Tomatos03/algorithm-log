# [153. 寻找旋转排序数组中的最小值](https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array/)

## 解题思路

1. 原转得到的数组有两种情况: 

   + 仍然是递增数组, 此时$nums[0]$就是最小值
   + 不是递增数组, 此时数组分为两段, 一段从$[0, i]$递增, 另一段从$[i + 1, n - 1]$递增, 并且满足$nums[i] > nums[i + 1] \and nums[0] > nums[n - 1]$


```cpp
// 时间复杂度O(log n)
class Solution {
public:
    int findMin(vector<int>& nums) {
        int n = nums.size();
        if (nums[0] <= nums[n - 1])
            return nums[0];

        nums.push_back(INT_MAX);
        int l = 0, r = n - 1;
        while (l <= r) {
            int mid = (l + r) >> 1;
            if (nums[mid] > nums[mid + 1]) {
                return nums[mid + 1];
            } else if (nums[0] <= nums[mid]) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return -1;
    }
};
```
