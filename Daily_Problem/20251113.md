# [3228. 将 1 移动到末尾的最大操作次数](https://leetcode.cn/problems/maximum-number-of-operations-to-move-ones-to-the-end/)

## 解题思路

1. 想要最大化操作次数, 必须从最左的$1$开始进行操作
2. 顺序遍历字符串, 对于当前回合的字符如果是$0$则跳过, 如果是$1$则统计连续$1$的个数
3. 每次操作对答案的贡献为当前回合连续$1$的个数$+$之前统计的$1$的个数

## 参考代码


```cpp
using i64 = long long;
using pii = pair<int, int>;
class Solution {
public:
    int maxOperations(string s) {
        int cnt = 0, i = 0, n = s.size(), pre_cnt = 0;
        while (i < n) {
            if (s[i] == '0') {
                ++i;
                continue;
            }

            int cur_cnt = 0;
            while (i < n && s[i] == '1') {
                cur_cnt++;
                ++i;
            }
            if (i < n)
                cnt += pre_cnt + cur_cnt;
            pre_cnt += cur_cnt;
        }
        return cnt;
    }
};
```
