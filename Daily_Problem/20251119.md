# [2154. 将找到的值乘以 2](https://leetcode.cn/problems/keep-multiplying-found-values-by-two/)

## 解题思路

1. 输入数据量很小, 可以考虑暴力枚举
1. 查找$original$是否在数组中需要$O(n)$的时间复杂度, 数组中的每一个数最多命中一次(即与$original$相等)需要$O(n)$的时间复杂度. 总时间复杂度为$O(n^2)$. 对于查找的过程可以使用二分查找优化到$O(\log n)$

## 参考代码


```cpp
// 时间复杂度O(n^2)
class Solution {
public:
    int findFinalValue(vector<int>& nums, int original) {
        int n = nums.size();
        bool is_find;
        do {
            is_find = false;
            for (int &v : nums) {
                if (v == original) {
                    original = original * 2;
                    is_find = true;
                    break;
                }
            }
        } while (is_find);
        return original;
    }
};
```
