# [3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

## 解题思路

1. 让$l = 0$表示子串区间左端点, 右端点$r$不断的向右扩展直到子串之中有重复的字符时停止. 此时以$l = 0$为起点的无重复字符的子串的最长长度为$r - l(不包含r, s[r] = s[k], l \le k < r)$

2. 注意到$[l, k]$不可能在通过与其他区间拼接得到更优的答案, 求解$[1, n]$字符串的无重复字符的最长子串能够转换为求解$[k + 1, n]$字符串的无重复字符的最长子串长度与$r - l$比较最大值

**注: 顺序遍历按照题设要求划分字符串: $[l_0, r_0]、[l_1, r_1] \cdots [l_k, r_k]$, 无重复字符的最长子串最多占有3个连续区间(其中两个区间不可能占满)**


```cpp
// 时间复杂度O(n)
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int ans = 0, n = s.size();
        if (n == 0)
            return 0;
        set<char> st;
        s = s + s[n - 1];
        for (int i = 0, j = 0; i <= n; ++i) {
            while (st.count(s[i])) {
                ans = max(ans, int(st.size()));
                st.erase(s[j]);
                ++j;
            }
            st.insert(s[i]);
        }
        return ans;
    }
};

```
