# [955. 删列造序 II](https://leetcode.cn/problems/delete-columns-to-make-sorted-ii/)

## 解题思路

1. 比较两个字符串找第一个$x[i] > y[i]$的位置需要时间复杂度$O(len(s))$
1. 顺序比较相邻字符串需要的时间复杂度为$O(n)$
1. 每当一个新位置加入到删除索引序列时会影响之前相邻字符串的字典序, 例如: $abc, bad, adb$


```cpp
// 时间复杂度O(n*len(s)^2)
class Solution {
    int comp(string &x, string &y, set<int> &st) {
        int n = x.size();
        for (int i = 0; i < n; i++) {
            if (st.count(i))
                continue;

            if (x[i] == y[i])
                continue;
            else if (x[i] > y[i])
                return i;
            else
                return -1;
        }
        return -1;
    }

public:
    int minDeletionSize(vector<string>& strs) {
        int i = 0, n = strs[0].size(), m = strs.size();
        set<int> st;
        while (i < n) {
            int j = 0;
            bool flag = true;
            while(j < m - 1) {
                int pos = comp(strs[j], strs[j + 1], st);
                if (pos != -1) {
                    st.insert(pos);
                    flag = false;
                    break;
                }
                j++;
            }

            if (flag)
                break;
            ++i;
        }
        return int(st.size());
    }
};
```
