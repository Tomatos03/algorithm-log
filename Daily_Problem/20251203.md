# [763. 划分字母区间](https://leetcode.cn/problems/partition-labels/)

## 解题思路

1. 注意到第一个区间的第一个字母总是确定的, 此时我只需要知道字母在字符串中最后一次出现的位置, 就能够确定同一字母仅出现在一个片段之中的最小区间. 这个区间确定之后第二个区间的第一个字母又是确定的. 以此类推可以得到最终的最优划分
1. 可以发现1中的划分步骤每一次得到的都是最小区间所以最后得到答案必然是最优的



## 参考代码


```cpp
// 时间复杂度O(n)
using i64 = long long;
class Solution {
public:
    vector<int> partitionLabels(string s) {
        int size = s.size();
        vector<int> mx_i(26);
        for (int i = 0; i < size; ++i) {
            int c = s[i] - 'a';
            mx_i[c] = i;
        }
        vector<int> ans;
        int i = 0;
        while (i < size) {
            int i_0 = i, mx_last = mx_i[s[i] - 'a'];
            while (i <= mx_last) {
                int c = s[i] - 'a';
                mx_last = max(mx_last, mx_i[c]);
                ++i;
            }

            ans.push_back(i - i_0);
        }
        return ans;
    }
};

```
