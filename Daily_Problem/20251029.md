# [3370. 仅含置位位的最小整数](https://leetcode.cn/problems/smallest-number-with-all-set-bits/)

## 解题思路

1. 找到给定整数$nums$的二进制表示中最高位$1$的位置$i$, 此时$\ge nums$且仅含置位位的最小整数为$(1 << i) - 1$

## 参考代码


```cpp
using i64 = long long;
using pii = pair<int, int>;
class Solution {
public:
    int smallestNumber(int n) {
        int i = 0;
        while (n > 0) {
            n >>= 1;
            ++i;
        }
        return (1 << i) - 1;
    }
};

```
